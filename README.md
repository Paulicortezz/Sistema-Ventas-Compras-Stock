Este proyecto implementa un sistema en modo consola para gestionar:
- Ventas y compras
- Stock
- Reportes

Incluye manejo de facturas con:
- Carga de cabecera
- Carga de detalle
- Cálculo de subtotal
- Pie de documento
- Cálculo de impuestos
- Cálculo de total
